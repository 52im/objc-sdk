# LeanCloud Objective-C SDK

## Features
  * [x] Data Storage
  * [x] Object Query
  * [x] Cloud Engine
  * [x] File Storage
  * [x] Short Message Service
  * [x] Push Notification
  * [x] Search Query
  * [x] Analytics (iOS only)
  * [x] Instant Messaging (iOS, macOS only)
